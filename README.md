# Сasino Bot


## Возожности

*   **Обязательная подписка:** Бот проверяет подписку на канал перед началом работы
*   **CryptoBot Интеграция:** 
    *   Пополнение баланса через CryptoBot (USDT, TON, BTC, ETC, LTC)
    *   Автоматический вывод средств через чеки (минимальная сумма вывода 100 RUB)
    *   Управление казной (просмотр и пополнение через админку)
    *   Реферальная система (1% от пополнений рефералов)
*   **Проиль пользователя:** Отображает ID, Username, Баланс и Дату регистрации
*   **Рассылка:** Удобная админ-панель для массовых рассылое
*   **Техподдержка:** Система тикетов с форумными топиками
*   **Производительность:** Асинхронная база данных (SQLite + aiosqlite)

### Игровые Режимы

1.  **Игры с ботом:**
    *   Слоты, Баскетбол, Боулинг, Дартс, Кубик, Футбол, Казино
    *   Настраиваемые коэффициенты выигрыша в админке
2.  **PvP Игры (Дуэли):**
    *   5 режимов: Кубик, Футбол, Баскетбол, Дартс, Боулинг
    *   Автоход при бездействии (60 сек)
3.  **Чат-игры в чате:**
    *   Поддержка игр прямо в чате через команды (/cub, /spin, /dice и др)
    *   Поддержка мультиплеера (/cub3p - 3 игрока, /cubtotal - игра на сумму очков, /cub3x - 3 раунда)

### Администрирование и Модерация

*   **Назначение модеров:** Команды /addmod, /delmod, /mods или через админку
*   **Управление играми:** Удаление игр командной /adelgame(модераторам)
*   **Розыгрыши (Фасты):**
    *   /fast [сумма] - создать быстрый розыгрыш от игрока
    *   /afast [сумма] [время в минутах] - создать закрепленный розыгрыш от администрации
    *   Автоматическое подведение итогов и выбор победителя



## Установка и Запуск

### 1. Подготовка
Установите **Python 3.10+**.

### 2. Зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройка config.py
```python
TOKEN = '...'           # Токен бота
ADMIN_ID = 123...       # ID админа /admin812
CHANNEL_ID = -100...    # ID Канала подписки
CHANNEL_LINK = '...'    # Ссылка на канал
SUPPORT_CHAT_ID = -100... # ID Группы с темами для ТП
CRYPTOBOT_TOKEN = '...'   # Токен CryptoBot
MAIN_CHAT_ID = -100...    # ID Чата для игр
```

**Важно для Техподдержки:**
1.  Создайте группу(чат)
2.  В настройках группы включите **"Темы" (Topics)**
3.  Добавьте бота в эту группу и выдайте права администратора (для создания тем и управления ими)
4.  Узнайте ID этой группы (обычно начинается с -100)

### 4. Запуск
```bash
python bot.py
```

## Команды для игр в чате:

**CLASSIC (классические игры):**
*   `/cub [сумма ставки]`
*   `/dar [сумма ставки]`
*   `/boul [сумма ставки]`
*   `/bas [сумма ставки]`
*   `/foot [сумма ставки]`

**X (игры до 2-5 побед):**
*   `/cub[2-5]x [сумма ставки]`
*   `/dar[2-5]x [сумма ставки]`
*   `/boul[2-5]x [сумма ставки]`
*   `/bas[2-5]x [сумма ставки]`
*   `/foot[2-5]x [сумма ставки]`

**TOTAL (игры по сумме 2-5 бросков):**
*   `/cubtotal[2-5] [сумма ставки]`
*   `/dartotal[2-5] [сумма ставки]`
*   `/boultotal[2-5] [сумма ставки]`
*   `/bastotal[2-5] [сумма ставки]`
*   `/foottotal[2-5] [сумма ставки]`

**3PLAYERS (игры на три человека):**
*   `/cub3p [сумма ставки]`
*   `/dar3p [сумма ставки]`
*   `/boul3p [сумма ставки]`
*   `/bas3p [сумма ставки]`
*   `/foot3p [сумма ставки]`

**OTHER GAMES:**
*   `/spin [сумма ставки]` - слоты в чате (выплата при выигрыше - 3Х)
*   `/dice [сумма ставки] [число]` - угадайте число на кубе (выплата при выигрыше - 2Х)

**Команды для взаимодействия в чате:**
*   `/del` [в ответ на сообщение с игрой] - удаление игры в чате
*   `/allgames` - выводит все активные игры чата
*   `/fast [сумма]` - создание быстрого розыгрыша между всеми игроками чата
*   `/getid` - выводит информацию о вашем аккаунте в боте
*   `/bal` - выводит ваш баланс в боте
*   `/admin812` - Админ-панель (в боте)
